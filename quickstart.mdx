Get up and running with Dart quickly by writing and running your first program.

## Prerequisites

Before you begin, [install the Dart SDK](/installation) on your system.

## Your first Dart program

<Steps>
  <Step title="Create a Dart file">
    Create a new file called `hello.dart`:

    ```dart hello.dart
    void main() {
      print('Hello, World!');
    }
    ```
  </Step>

  <Step title="Run the program">
    Execute your Dart program using the `dart` command:

    ```bash
    dart hello.dart
    ```

    You'll see the output:

    ```
    Hello, World!
    ```
  </Step>
</Steps>

## Working with arguments

Dart makes it easy to work with command-line arguments:

```dart greet.dart
void main(List<String> arguments) {
  if (arguments.isEmpty) {
    print('Usage: dart greet.dart <name>');
    return;
  }
  
  final name = arguments.first;
  print('Hello, $name!');
}
```

Run it with:

```bash
dart greet.dart Alice
# Output: Hello, Alice!
```

## Language basics

### Variables and types

Dart is strongly typed with type inference and null safety:

```dart
void main() {
  // Type inference
  var name = 'Alice';
  var age = 30;
  
  // Explicit types
  String city = 'New York';
  int? nullableAge;  // Can be null
  
  // Null safety
  print(nullableAge ?? 0);  // Use 0 if null
}
```

### Collections

Work with lists, sets, and maps:

```dart
void main() {
  // Lists
  var numbers = [1, 2, 3, 4, 5];
  numbers.add(6);
  
  // Sets
  var uniqueNames = {'Alice', 'Bob', 'Alice'};  // Only 2 items
  
  // Maps
  var person = {
    'name': 'Alice',
    'age': 30,
    'city': 'New York',
  };
  
  print(person['name']);  // Alice
}
```

### Functions

Dart supports multiple function styles:

```dart
// Regular function
int add(int a, int b) {
  return a + b;
}

// Arrow function
int multiply(int a, int b) => a * b;

// Named parameters
void greet({required String name, String greeting = 'Hello'}) {
  print('$greeting, $name!');
}

void main() {
  print(add(2, 3));        // 5
  print(multiply(4, 5));   // 20
  greet(name: 'Alice');    // Hello, Alice!
}
```

### Control flow

Use modern control flow with pattern matching:

```dart
void main() {
  var score = 85;
  
  // if-else
  if (score >= 90) {
    print('A');
  } else if (score >= 80) {
    print('B');
  } else {
    print('C');
  }
  
  // Switch expressions
  var grade = switch (score) {
    >= 90 => 'A',
    >= 80 => 'B',
    >= 70 => 'C',
    _ => 'F'
  };
  print(grade);  // B
}
```

## Asynchronous programming

Dart has first-class support for async/await:

```dart
import 'dart:async';

Future<String> fetchUserData() async {
  // Simulate network delay
  await Future.delayed(Duration(seconds: 1));
  return 'User data loaded';
}

void main() async {
  print('Fetching user data...');
  var data = await fetchUserData();
  print(data);  // User data loaded (after 1 second)
}
```

### Working with Streams

Handle multiple asynchronous values:

```dart
Stream<int> countStream(int max) async* {
  for (int i = 1; i <= max; i++) {
    await Future.delayed(Duration(milliseconds: 100));
    yield i;
  }
}

void main() async {
  await for (var number in countStream(5)) {
    print(number);  // Prints 1, 2, 3, 4, 5 with delays
  }
}
```

## Creating a project

Use `dart create` to set up a new project:

<Tabs>
  <Tab title="Console app">
    ```bash
    dart create -t console my_app
    cd my_app
    dart run
    ```
  </Tab>
  
  <Tab title="Web app">
    ```bash
    dart create -t web my_app
    cd my_app
    dart run
    ```
  </Tab>
  
  <Tab title="Package">
    ```bash
    dart create -t package my_package
    cd my_package
    dart test
    ```
  </Tab>
</Tabs>

## Compiling to native code

Compile your Dart code to a standalone executable:

```bash
dart compile exe hello.dart -o hello
./hello  # Run the native binary
```

<Info>
Native executables start instantly and don't require the Dart SDK to run.
</Info>
